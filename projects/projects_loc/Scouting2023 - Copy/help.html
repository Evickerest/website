Instructions:

Step One:

First select the "Info" tab.

Then, type in your name to "Scouter Name", the current match number into "Match #" and the robot team number you are scouting into "Team #".

If at any point the robot breaks, and cannot compete during the match in any meaningful way, you can select "robot broke".

Step Two:

Second, select the "Auto" tab.

Then, select what the robot does in auto.

  • Engaged: The robot is balanced and all other robots contacting the charge station are docked
  • Docked: The robot is contacting the charge station.
  • None: If the robot does not do anything, select "none".
  • Mobility: The robot leaves their community during auto.

Note: It is possible to be engaged with the charge station but not get mobility points during auto.

Then, select the points scored during auto.

If the robot scores a game piece, press the "+" button in the "Hits" column on either High, Mid, or Low depending on where it was scored.

If the robot attempts to score a game piece and fails, press the "+" button in the "Misses" column on either High, Mid, or Low depending on where it was attempted.

If at any point the "+" button was incorrectly pressed, you can press the "-" button to take away 1 from the value.

Step Three:

Third, select the "Teleop" tab.

This section is identical to what was in auto.

Step Four:

Lastly, select the "Endgame" tab.

Select what the robot does in the endgame.

  • Engaged: The robot is balanced and all other robots contacting the charge station are docked
  • Docked: The robot is contacting the charge station.
  • Parked: The robot ends in their community at the end of teleop.
  • None: If the robot does not do anything, select "none".

Note: If a robot is docked or engaged, that is not considered parked.

Then, select what happens on the charge station.

  • Double Engaged: If two robots dock and engage with the charge station.
  • Triple Engaged: If three robots dock and engage with the charge station.
  • Attempted Double/Triple: If two or three robots try to enagage and fail.

Then, give a rating to the following descriptions:

  • Cycle Speed: This represents how fast the robot can pick up a piece, score it, and go get another piece.
  • Maneuverability: This represents how well the robot can traverse the field, including how well they can dodge other robots.
  • Defensive Ability: This represents how well a robot can play defense. If the robot did not play defense, leave this rating blank.
  • Overall Rating: This represents how well the scouter thinks the robot did in comparision to other robots.

Then, write in any comments. These are for other points that the scouter finds important.

Step Five:

Press submit!!!

If pressing submit does not work, it may mean that not everything was filled out. Certain fields are required to be filled out in order to submit. However, this is not the case if you select the "robot broke" checkbox.

If all goes well, it will says "Submission Succesful!"
